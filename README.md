# 📊 Customer Behavior Analysis Dashboard

An end-to-end data analysis project that explores customer shopping behavior using **Python, SQL, and Power BI**. The project focuses on extracting actionable insights from customer transaction data to support data-driven business decisions.

---

## 🚀 Project Overview

This project analyzes customer shopping data to understand:
- Customer purchasing behavior and trends  
- Revenue distribution across demographics  
- Product performance and customer satisfaction  
- Discount effectiveness and usage patterns  
- Subscription and retention behavior  

---

## 🛠️ Tools & Technologies

- **Python (Pandas, NumPy)** – Data cleaning, preprocessing, and exploratory data analysis  
- **SQL** – Analytical queries, aggregations, and segmentation  
- **Power BI** – Interactive dashboard and visualization  
- **Excel/CSV** – Data source

## 🔍 Key Analyses Performed

## 🔄 Data Processing Pipeline

### 🧹 Python (Data Cleaning & Feature Engineering)
- Handled missing values using median-based imputation  
- Standardized column names and formats  
- Created new features:
  - Age group segmentation  
  - Purchase frequency conversion  

### 🧠 SQL (Analysis & Insights)
- Aggregation queries for revenue metrics  
- Customer segmentation using conditional logic  
- Discount and product performance analysis  
- Use of CTEs for advanced queries  

---

## 💡 Key Insights

- Majority of customers are non-subscribers (~73%)  
- Middle-aged customers contribute highest revenue  
- Clothing and accessories are top-performing categories  
- Discount usage significantly impacts purchasing behavior
