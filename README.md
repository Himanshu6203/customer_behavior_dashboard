# Customer Behavior Analysis

An end-to-end data analysis project that explores and analyzes customer shopping behavior patterns. This project combines Python data cleaning and exploration with SQL-based analytical queries to uncover actionable insights about customer purchasing habits, preferences, and segmentation.

## 📋 Project Overview

This project analyzes customer shopping data to answer critical business questions around:
- Revenue patterns across customer demographics
- Product performance and customer ratings
- Discount effectiveness and usage patterns
- Customer segmentation and subscription behavior
- Shipping preferences and their impact on spending
- Purchase frequency and customer lifecycle classification

## 🛠️ Tools & Technologies

- **Python**: Data loading, cleaning, transformation, and exploratory data analysis using Pandas
- **SQL**: Advanced analytical queries, aggregations, and customer segmentation
- **Data Format**: CSV dataset with customer transaction records

## 🔍 Key Questions Analyzed

1. **Revenue by Gender** - Total revenue generated by male vs. female customers
2. **Discount & Spending** - Which customers used discounts but spent above average
3. **Product Performance** - Top 5 products by average review rating
4. **Shipping Impact** - Comparison of average purchase amounts between shipping types
5. **Subscription Analysis** - Do subscribed customers spend more than non-subscribers?
6. **Discount Trends** - Products with highest percentage of discounted purchases
7. **Customer Segmentation** - Classification into New, Returning, and Loyal customers
8. **Category Performance** - Top 3 most purchased products within each category
9. **Repeat Buyer Subscriptions** - Likelihood of repeat buyers (>5 purchases) to subscribe
10. **Age Group Revenue** - Revenue contribution and spending patterns by age group

## 📊 Data Processing Pipeline

### Python (Data Cleaning & Exploration)
- Load customer shopping behavior dataset
- Exploratory data analysis (head, info, describe)
- Missing value detection and imputation (median-based by category)
- Column standardization (lowercase, underscores, naming conventions)
- Feature engineering:
  - Age group segmentation (Young Adult, Adult, Middle-aged, Senior)
  - Purchase frequency conversion (text to days)

### SQL (Analysis & Insights)
- Aggregation queries for revenue metrics
- Window functions for ranking and segmentation
- Conditional aggregations for discount analysis
- CTE (Common Table Expressions) for complex segmentation logic
